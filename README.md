## 구현한 기능

> 로그인 / 로그아웃  
> 회원가입 및 팔로잉  
> 게시글 업로드(사진) / 수정 / 삭제  
> 게시글에 대한 좋아요와 댓글, 대댓글  
> 친구 목록  
> 친구 추천  
> 1:1 채팅  
> 유저 페이지 / 마이 페이지
